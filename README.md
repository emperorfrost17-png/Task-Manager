# Task Manager

A simple, clean task management web app built with vanilla HTML, CSS, and JavaScript. Create, organize, and track your daily tasks right in the browser — no frameworks, no build step.

🔗 **Live Demo:** [https://emperorfrost17-png.github.io/Task-Manager/](https://emperorfrost17-png.github.io/Task-Manager/)

## Features

- ✅ Add and delete tasks
- ✏️ Edit existing tasks
- ☑️ Mark tasks as complete
- 📅 Set due dates
- 🔥 Set priority levels (High / Medium / Low)
- 🔍 Filter or search through tasks
- 🏷️ Organize tasks with categories/tags
- 💾 Tasks persist after refresh (saved with `localStorage`)

## Tech Stack

- **HTML** – structure and layout
- **CSS** – styling and responsiveness
- **JavaScript** – task logic and interactivity

## Getting Started

No installation or dependencies required — it's a static site.

1. Clone the repository
   ```bash
   git clone https://github.com/emperorfrost17-png/Task-Manager.git
   ```
2. Open the project folder
   ```bash
   cd Task-Manager
   ```
3. Open `index.html` in your browser (or use a tool like VS Code's Live Server extension)

That's it — you're up and running!

## Usage

- Type a task into the input field and hit **Add** to create it.
- Click a task to mark it complete, or use the edit button to update its details.
- Use the priority and category options to keep tasks organized.
- Use the search/filter bar to quickly find specific tasks.
- Your tasks are saved automatically in your browser, so they'll still be there the next time you open the app.
